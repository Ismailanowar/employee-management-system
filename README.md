# Employee Management System

A modern Employee Management System built with Next.js, TypeScript, Tailwind CSS, Prisma, PostgreSQL, and Auth.js.

## Features

- Authentication
- Dashboard
- Employee CRUD
- Department CRUD
- Search
- Filter
- Pagination
- Responsive Design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- Auth.js
